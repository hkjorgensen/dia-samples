DIA Samples

Here is the list of samples. These links - and most of the demos themselves - expect that you are accessing them via your Kattegat server, eg, from http://localhost:3000/demos.html.

Basics

square

Basic text manipulation, events, creating HTML elements, assigning CSS properties and transitioning CSS values.

square-store

Storing and fetch data in the browser or on the server

square-decouple

Controlling the square creation from another page - or potentially another server. Demonstrates Kattegat's realtime communication capabilities

square-direct

Demonstrates how to make a more direct interaction, using more keyboard events and fancier CSS.

Touch and Pointers

pointer

Setting CSS properties on the basis of pointer up and down events, and also how to track whether the pointer was held down

pointer-drag

Visualises the pointer position, and shows how to move an element around to match the pointer

pointer-playground

Play with pointer events and see what data the event holds

pointer-world

Broadcast pointermove events to other windows/devices

touch-swipe

Change colour based on the velocity of swipe gesture

touch-solar

Create and manipulate HTML elements based on touch events and gestures

touch-playground

Play with touch gesture events and see what data the event holds

touch-mix

Mix colour and transform an element from different devices

Audio

audio-play

Play a sample and sequence synchronised sample playback

audio-filter

Loop a sample and filter the sound based on pointer movement

audio-pan

Start one or more loops and change how they are panned in the soundscape

audio-play

Manipulate frequency and volume with a pad

audio-filter

Use sound as feedback for distance between two object

audio-pan

Execute commands with your voice

speech-outout

Use text-to-speech
